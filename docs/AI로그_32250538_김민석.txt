1. 추가되었으면 하는 통계 지표가 있나요? 에이, 말씀하신 구질 무브먼트니 뭐니 하는 건 저한테 너무 어려워요. 저는 그런 전문 용어는 잘 모르겠고... 그냥 단순하게 이 선수가 '요즘' 잘하는지, 아니면 '예전'보다 실력이 많이 죽었는지가 궁금해요. 4~5년 전이랑 비교해서 타율이 얼마나 떨어졌는지, 아니면 나이가 들어서 발이 느려졌는지 같은 거요. 복잡한 지표보다는 그냥 딱 보고 "아, 이 친구 예전만 못하네" 아니면 "아직 팔팔하네"라고 바로 알 수 있는 수준이면 좋겠어요.
2. 데이터를 엑셀(CSV)이나 이미지로 내보내는 기능이 필요하신가요? 아이고, 엑셀 같은 건 쓸 줄도 모르고 쓸 일도 없어요. 이미지로 저장하는 건... 가끔 친구들한테 카톡으로 "얘 봐라, 요새 장난 아니다" 하고 보여줄 때 쓸 수도 있겠는데, 사실 그것도 캡처하는 게 일이라 잘 안 할 것 같아요. 저 같은 사람한테는 그런 복잡한 기능은 크게 의미가 없네요.
3. 선수 간의 기록을 1:1로 비교해주는 '비교하기' 기능이 있다면 얼마나 자주 사용하실 것 같나요? 글쎄요, 가끔 뉴스에서 "누가 이번에 연봉을 더 많이 받네" 이런 소리 들리면 "도대체 누가 더 잘하길래?" 싶어서 한두 번 찾아볼 수는 있겠죠. 근데 그걸 매번 들어가서 비교해보고 있지는 않을 것 같아요. 제가 뭐 데이터 분석하는 사람도 아니고, 그냥 궁금할 때 한두 번 슥 보는 정도 아닐까요?
4. 선수나 팀의 기록을 검색할 때 과정이 직관적인게 좋다고 생각하나요? 당연하죠. 저는 뭐 하나 찾으려고 버튼 여러 번 누르고, 영어로 된 거 해석해야 하고 이러면 그냥 창 닫아버려요. 유튜브 검색하는 것처럼 그냥 이름만 딱 치면 바로 나오고, 복잡하게 안 들어가도 한 화면에 중요한 건 다 보였으면 좋겠어요. 눈도 침침한데 글자 작고 복잡한 건 딱 질색이에요.
5. 정보 업데이트 속도는 어느정도였으면 좋겠나요? 저는 경기를 실시간으로 계속 보고 있는 게 아니라서, 어제 경기 결과가 오늘 아침에 뉴스 볼 때 정확하게 반영되어 있기만 하면 돼요. 사실 일주일 단위로 몰아서 보기도 하니까, 아주 초 단위로 빠를 필요까지는 없는데... 그래도 내가 찾아봤을 때 지난 경기 기록이 안 올라와 있으면 좀 답답하긴 하겠죠. 적어도 다음 날 아침에는 정리가 싹 되어 있어야 보지 않겠어요?
